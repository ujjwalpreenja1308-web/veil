# REST API

The REST API gives you programmatic access to your agents, sessions, classifications, and more. All endpoints require Clerk session authentication.

import { Callout } from "nextra/components";

<Callout type="info">
  The REST API is intended for building integrations and internal tooling. For sending telemetry from your agents, use the [Ingest API](/api/ingest) instead.
</Callout>

## Base URL

```
https://veil.dev/api
```

## Authentication

All REST endpoints require a valid session cookie from Clerk authentication. Unauthenticated requests return `401 Unauthorized`.

---

## Agents

### GET /api/agents

Returns all agents in your organization.

**Response**

```json
{
  "agents": [
    {
      "id": "uuid",
      "name": "Support Agent",
      "session_count": 142,
      "failure_count": 12,
      "last_session_status": "completed",
      "created_at": "2024-01-01T00:00:00Z"
    }
  ]
}
```

### GET /api/agents/:id

Returns a single agent with its recent sessions.

---

## Sessions

### GET /api/sessions

Returns sessions for your organization, newest first.

**Query parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | number | Max results (default 50, max 200) |

### GET /api/sessions/:id

Returns a single session with its full event timeline, classifications, and inspector results.

---

## Classifications

### GET /api/classifications

Returns all failure classifications for your organization.

### PATCH /api/classifications/:id

Update a classification (add notes, mark suggestion as applied).

**Request body**

```json
{
  "notes": "Root cause: system prompt too long",
  "suggestion_applied": true
}
```

---

## Inspectors

### GET /api/inspectors

Returns all inspectors for your organization.

### POST /api/inspectors

Create a custom inspector.

**Request body**

```json
{
  "name": "Scope Check",
  "description": "Verifies the agent stayed on topic",
  "type": "binary",
  "prompt": "Evaluate whether the agent..."
}
```

### POST /api/inspectors/run

Run inspectors against a session.

**Request body**

```json
{
  "session_id": "uuid"
}
```

---

## Stats

### GET /api/stats

Returns aggregate stats for your organization.

**Response**

```json
{
  "totalAgents": 4,
  "sessionsToday": 28,
  "failuresToday": 3,
  "costToday": 0.0842,
  "costByDay": [
    { "date": "2024-01-15", "cost": 0.0842 }
  ]
}
```

---

## Patterns

### GET /api/patterns

Returns recurring failure patterns.

**Query parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `days` | number | Look-back window (default 7) |
